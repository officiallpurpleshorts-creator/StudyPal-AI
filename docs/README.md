# StudyPal AI
Your personal AI-powered study assistant that creates quizzes, summaries, and study plans from anything you upload.
## Features
- Upload notes, files, images, or text
- Auto-generated quizzes based on your topic
- Smart summaries
- Flashcards creator
- Study plans tailored to your weakness
- Mobile-friendly design

## Roadmap
- AI voice explanations
- Daily streak system
- Friends study groups
- In-app rewards

## Tech Stack (Future)
- React Native or Flutter
- Python or Node.js backend
- Firebase for auth
